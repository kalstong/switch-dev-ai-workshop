# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is an AI workshop presentation repository — a static HTML slide deck with accompanying presenter notes. There is no build system, no dependencies, and no tests.

## Development

Serve the slides locally:

```
npx serve .
```

Then open `slides.html` in the browser. The launch.json configuration (`.claude/launch.json`) is pre-configured with `npx serve` for Claude Code preview.

## Architecture

- **slides.html** — Self-contained HTML slide deck (CSS + JS inlined). Blueprint-themed presentation on AI for developers. Slides are navigated with arrow keys, spacebar, or swipe. Each slide is a `<div class="slide">` inside `<div class="deck">`. Adding a slide requires updating the total count in each slide's sheet number and adding a nav dot (auto-generated by JS based on slide count).
- **presenter-notes.md** — Talking points per slide, structured as markdown sections matching slide order.

## Conventions

- The slide deck is fully self-contained — no external dependencies, CDN links, or build steps. All CSS and JS must remain inlined in the HTML file.
- The visual theme is architectural blueprint (deep navy background, monospace headings, grid overlay, dimension lines, technical annotations). Maintain this aesthetic when adding or modifying slides.
- CSS custom properties are defined on `:root` (prefixed `--bp-*`). Use these for all color and font references.
