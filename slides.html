What is an LLM?

Large Language Models are neural networks trained on vast text corpora to predict and generate human language.

  • Scale — Billions of parameters encoding language patterns
  • Training — Learns from books, code, articles, conversations
  • Output — Generates text one token at a time, probabilistically
  • Capability — Reasoning, coding, analysis, creative writing

Prompt Engineering

The art of communicating with LLMs effectively.

Techniques
  • Zero-shot — Direct question, no examples
  • Few-shot — Provide examples to guide format
  • Chain-of-thought — "Think step by step"
  • System prompts — Set behavior and persona
Best practices
  • Be specific about the output format
  • Provide context and constraints
  • Break complex tasks into steps
  • Iterate and refine your prompts

Section C

Skills & Commands

Skills

Packaged capabilities an agent can invoke on demand. A skill bundles instructions, tools, and context into a reusable unit.

  • Code review
  • Documentation generation
  • Data visualization
  • Custom domain skills
Commands

Slash commands are user-facing shortcuts that trigger specific workflows or modes within the agent.

  • /init — Bootstrap a project
  • /review — Code review
  • /help — Usage guide
  • /config — Settings
